# Jonny Shannon — Email Signature

Email signature for Jonny Shannon (Award Winning Speaker, Author & Youth Worker).

## Files
- **jonny-signature.html** — production signature. Image served via jsDelivr CDN
  (`https://cdn.jsdelivr.net/gh/JonnyShan/jonnyshannon-signature@main/jonny-headshot-badges.png`). Works in Gmail, Outlook, Apple Mail, iOS.
- **jonny-signature-preview.html** — self-contained (image embedded as base64).
  Renders standalone in a browser / Apple Mail without internet hosting.
- **jonny-headshot-badges.png** — headshot + award badges composite (197×224).

## Install (Gmail)
1. Open `jonny-signature.html` in a browser.
2. Select all (⌘A) → copy (⌘C).
3. Gmail → Settings → General → Signature → paste.

## Install (Apple Mail / iOS)
Use `jonny-signature-preview.html` the same way, or paste the production file.
